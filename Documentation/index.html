Introduction


Ubiquitous - Social Network Aggregator HTML Template

Ubiquitous is a Social Network Aggregator HTML template, using most popular design trends as HTML5, CSS3 and jQuery. The template have fullpage scroll function and separate panel sections, so you can create various styles of page to suit your needs.

Social network aggregation is the process of collecting content from multiple social network services into one unified presentation. Ubiquitous aside from simplifying the user's social networking experiences. The template designed to help companies and individuals improve engagement with their brands by creating aggregated social streams that can be embedded into an existing website and customized to look visually intrinsic to the site. This allows potential customers to interact with all the social media posts maintained by the brand without requiring them to jump from site to site. This has the benefit of keeping customers on the brand's site for a longer period of time.



Configuration and Installation

Main-Package zip: Main zip file HTML and Documentation Included

(1) Main and (2) Documentation folder

Open the "Main" folder and run index.html in your browser

  1. Open the Main Folder to find all the Templates Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • Main/assets/css - Extra Stylesheets Folder
    • Main/assets/js - Javacripts Folder
    • Main/index.html - Index File/Homepage
    • Main/config.json - Main Configuration File
    • Main/about.html - Text Used by About Section
    • Main/widget-instagram.html - Widget Used by Instagram Section ( if no accessToken exists )
    The other files can be used according to your preferences.
  4. You're now good to go..! Edit main coniguration file (config.json), fill it with your social network accounts and show off your Brand New Beautiful Website in style.

Instagram Section


Instagram Section is a simple way to display your Instagram photos on your website, powered by the new Instagram Basic Display API.

Setting up Instagram is pretty straight-forward - there are 2 main steps.

  1. Create a Facebook app linked to Instagram, and add yourself as a test user. See Managing User Tokens.
  2. Create an access token and provide it to an Instagram Token service

If you don't want to use the Instagram API, you can use any free service to create an HTML widget.

Sample widget widget-instagram.html was generated by Powr.io service.

If the access Token field remains empty, a file is loaded from the widget field.

    "instagram": {
    "title": "Instagram",
    "account": "product.daily",
    "count": 9,
    "accessToken": "",
    "widget": "widget-instagram.html"
    },

Send Email from a Static HTML Form using Google Apps Mail!

A Step-by-Step Example of using an HTML Form to send a "Contact Us" Message via Email without a Backend Server using a Google Script - No PHP, Python, Ruby, Java, Node.js etc.

See a working example here: https://dwyl.github.io/learn-to-send-email-via-google-script-html-no-server/

Why?

We needed a way of sending an email from a "static" HTML page when you don't (want to) have a server.

Key Advantages

No "Backend" to Deploy/Maintain/Pay for Fully Customisable - every aspect is customisable! Email sent via Google Mail which is Whitelisted Everywhere (high deliverability success) Collect/Store any form data in a Spreadsheet for easy viewing (perfect if you need to share it with non-technical people). Instead of using a server to send your email, which is easy but requires maintenance, use Google to send mail on your behalf and use Google Spreadsheets to keep track of the data!

If you prefer a standard server script for sending messages, there is an example of this script included "mail.php", in which you need to edit the recipient's Email address.
